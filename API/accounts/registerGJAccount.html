注册账号 #

接口 #

请求方式路径
POST/accounts/registerGJAccount.php

参数 #

名称说明
userName用户名
password密码
email邮箱
secretWmfv3899gc9

返回值 #

内容说明
1注册成功
-1注册失败
-2用户名已被使用
-3邮箱已被使用
-4用户名无效
-5密码无效
-6邮箱无效
-7密码不匹配
-8密码少于 6 个字符
-9用户名少于 3 个字符
-99邮箱不匹配

示例 #

POST /accounts/registerGJAccount.php
Content-Type: application/x-www-form-urlencoded

userName=Homo&password=1145141919810&email=114514@1919810.com&secret=Wmfv3899gc9

响应 #

1
